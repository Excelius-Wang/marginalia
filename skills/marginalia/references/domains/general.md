# 领域 Profile：General（默认）

判断不出明确领域，或论文跨领域时，用这个默认 profile。

## Scope / Subfield

写论文真正的小研究方向（一句话），不要堆关键词。例：`语言模型预训练`、`图神经网络可扩展性`、`因果推断方法`、`分布式训练系统`。

## Tags（通用词表，按论文实际取 3–8 个）

`Method`、`Empirical Study`、`Theory`、`Benchmark`、`Dataset`、`System`、`Survey`、`Measurement`、`Optimization`、`Evaluation`、`Reproducibility`、`Application`。

## 领域专用批判提示

- baseline 是否公平、是否最新？
- 指标是否真的衡量了 claim 关心的东西？
- 数据集规模 / 代表性是否支撑结论声称的泛化范围？
- 结果是否报告了方差 / 多次运行 / 显著性？
